現在の状況
米ドル指数（DXY）は98.80で、MA50（98.78）およびMA200（98.52）をわずかに上回る水準に位置し、テクニカル上は辛うじてトレンド維持。RSI14は49.64と中立圏で方向感を欠く。フェデラルファンズ金利3.64%に対し米10年債利回りは4.34%と逆イールドは解消、10-2スプレッドは+0.53%でカーブは正常化している。一方でGDP前期比年率は0.48%と極めて弱く、失業率は4.3%に上昇、CPI前年比は3.32%と目標水準を依然上回るスタグフレーション的環境が続く。VIX9d（18.04）に対しVIX3m（21.48）が高く、中期的な不確実性が短期より高い状態にある。なお、AAIIサーベイデータは403エラーで取得不能のため、センチメント評価はVIX・プットコール・SKEWに依拠する。
強気の論点
  • claim: 高い金利水準がドルの対外キャリー優位を支える / data: {'fed_funds': 3.64, 'us10y': 4.34} / evidence: フェデラルファンズ金利3.64%・米10年債4.34%は主要先進国との金利差を維持しており、ドル建て資産への資金流入を促しやすい状況が続く。
  • claim: テクニカル上は短中期の移動平均を上回り下値サポートが機能 / data: {'price': 98.8, 'ma50': 98.78, 'ma200': 98.52} / evidence: DXYはMA50（98.78）・MA200（98.52）双方を上回っており、両均線が緩やかなサポートとして機能している。価格がこれらを割り込んでいない間はトレンド崩壊とは言えない。
  • claim: 正常なイールドカーブは市場の景気悲観論後退を示す / data: {'yield_spread_10_2': 0.53, 'us2y': 3.83, 'us10y': 4.34} / evidence: 10-2年スプレッドが+0.53%と正の領域にあり、深刻な景気後退がすぐに織り込まれている状態ではない。ドル売りの最大動機である急速なFed利下げ期待が必ずしも支配的ではない可能性。
  • claim: SKEW高水準がテール保険需要＝ドル安全資産買いを示唆 / data: {'skew': 139.59} / evidence: SKEW指数が139.59と高水準にあり、市場参加者が極端なリスクシナリオに備えた保険購入を行っていることを示す。リスクオフ局面でのドル安全資産需要が潜在的に存在する。
弱気の論点
  • claim: 準ゼロ成長がFedの早期利下げを迫り、金利優位を侵食するリスク / data: {'gdp_qoq_annualized': 0.48, 'fed_funds': 3.64} / evidence: GDP前期比年率0.48%は景気後退寸前の水準であり、雇用・消費の悪化が続けばFedは想定以上の利下げを余儀なくされる。利下げはドルの金利差優位を直接圧縮する。
  • claim: 失業率の上昇がスタグフレーション懸念を強化し、ドルの信認を損なう / data: {'unrate': 4.3, 'cpi_yoy': 3.32} / evidence: 失業率4.3%はFedのデュアルマンデートで雇用悪化サイドを意識させる水準であり、高インフレ（CPI 3.32%）との同時進行はスタグフレーションシナリオへの不安を高め、USD売り圧力となりうる。
  • claim: VIXタームストラクチャーが中期リスク上昇を示唆し、ドルのモメンタムを阻害 / data: {'vix': 19.31, 'vix3m': 21.48} / evidence: VIX3m（21.48）がVIX（19.31）を大きく上回っており、短期的な落ち着きの裏で中期的な不確実性が高い。リスク再燃局面ではドルが追い風を受ける場合もあるが、資本逃避が不規則なドル変動を生む懸念がある。
  • claim: RSIが中立付近で方向感なく、上値追いの勢いが欠如 / data: {'rsi14': 49.64, 'price': 98.8, 'ma200': 98.52} / evidence: RSI14が49.64と50をわずかに下回る水準にあり、強気・弱気いずれへの転換も示していない。モメンタム不在はブレイクアウト局面での下方リスクが上方リスクより先に顕在化しやすいことを意味する。
総合判断
DXYは移動平均線を辛うじて上回るものの、GDP前期比年率0.48%というほぼ停滞に近い成長、失業率4.3%への上昇、およびCPI3.32%の高止まりが重なるスタグフレーション的環境が最大のリスク。Fedが成長悪化を優先して利下げを加速すれば、現在の金利差優位は急速に縮小する。一方でイールドカーブ正常化とSKEW高水準が示すテール保険需要はドルの急落を抑制する緩衝材となりうる。総合すると、率差サポートが残るも成長・雇用の弱さがじわじわと侵食するシナリオを中心に置き、わずかにベアリッシュな評価が妥当と判断する。
主なリスク
  • CPIが再加速し3.5%超を維持、Fedが利上げ再開または長期据え置きを明示してドル金利優位が強化される
  • 主要競合国（ECB・BOJ等）の景気悪化が米国より深刻となり、相対的なドル選好が強まる
  • 地政学的ショックによるリスクオフが急激に強まり、ドルへの安全資産需要が一時的に急増する
  • 米GDP Q2データが急反発し、スタグフレーション懸念が払拭される
  • DXY技術的サポート（MA200付近98.52）が強く機能し、ダウントレンドが否定されるダブルボトム形成
主な触媒（カタリスト）
  • 米GDP Q2が2期連続0%台もしくはマイナス成長となり、景気後退確定でFed利下げが加速する
  • 失業率が4.5%以上に上昇し、Fedが緊急的な利下げ姿勢に転換する
  • DXYがMA200（98.52）を明確に割り込み、テクニカルな売りシグナルが連鎖する
  • Fedが次回FOMCで利下げを実施し、タカ派ガイダンスを後退させる
  • CPI鈍化データが続き、インフレ鎮静化と成長低迷が同時進行するディスインフレ局面入りが確認される
