現在の状況
米ドル指数（DXY）は2026年5月23日時点で99.19と、50日移動平均（98.98）および200日移動平均（98.55）の双方を上回り、前週比でテクニカル的な改善が確認された。RSI14は56.83と中立域のやや上位に位置し、方向感はやや強気に傾いている。FFレートは3.62%、米2年債利回り4.08%・10年債利回り4.57%で10年-2年スプレッドは+0.43%と正の傾きを維持。CPIは前年比3.95%とFRB目標の約2倍で高止まりし、GDP前期比年率は1.99%と潜在成長率付近で推移。ミシガン大消費者信頼感（UMich代替指標）は49.8と典型レンジ（50-110）の下限を割り込み、前週の53.3からさらに悪化している。
強気の論点
  • claim: DXYが50日MA・200日MAを双方上回りテクニカルモメンタムが好転 / data: {'price': 99.19, 'ma50': 98.976, 'ma200': 98.547} / evidence: 前週まで50日MAをわずかに下回りベアポイントとされていたDXYが99.19と50日MA（98.98）・200日MA（98.55）を明確に上回った。短期・長期双方のテクニカルサポートが同時に機能している状態は、ドルの上昇継続を示すポジティブシグナルである。
  • claim: CPI高止まりによるFRBタカ派バイアスが金利差優位を維持 / data: {'cpi_yoy': 3.947, 'fed_funds': 3.62} / evidence: CPIが前年比3.95%とFRB目標2%の約2倍で推移しており、FRBが積極的な利下げに転じられない環境が継続している。FFレート3.62%という高水準が対主要通貨との金利差優位を維持し、ドルへの資本流入をサポートし続けている。
  • claim: 米長期金利の上昇がドル建て資産の投資魅力を向上 / data: {'us10y': 4.57, 'us2y': 4.08, 'yield_spread_10_2': 0.43} / evidence: 米10年債利回りが4.57%と前週の4.47%から約10bp上昇し、グローバルな資本が米ドル建て資産に引き寄せられる構造が強まった。2年債も4.08%と前週4.00%から上昇しており、短中期の金利優位も拡大している。
  • claim: 近接VIXの低水準が市場の安定を示しドルの短期的な乱高下リスクを低減 / data: {'vix9d': 14.08, 'vix': 16.76} / evidence: VIX9dが14.08、スポットVIXが16.76と双方低水準にあり、近接の市場不安は極めて乏しい。リスクオフに伴う急激なボラティリティ上昇がドルを不安定化させるリスクが低い環境はドルの安定推移を支持する。
弱気の論点
  • claim: ミシガン大消費者信頼感が典型レンジ下限を割り込み内需鈍化リスクが台頭 / data: {'aaii_bull_pct_umcsent': 49.8} / evidence: UMich消費者信頼感指数（aaii_bull_pct代替）が49.8と典型レンジ（50-110）の下限を割り込んでいる。前週53.3からさらに悪化しており、消費者マインドの急激な劣化は米国内需鈍化を示唆し、ドルのファンダメンタルズ悪化につながりうる。
  • claim: VIX先物のコンタンゴ構造が中期的な市場リスク拡大を示唆 / data: {'vix3m': 20.0, 'vix': 16.76, 'vix9d': 14.08} / evidence: VIX3m（20.0）がスポットVIX（16.76）を3.24pt上回るコンタンゴ構造が継続している。近接の安定と中期の不確実性の乖離は、数ヶ月以内にボラティリティが上昇するリスクをマーケットが折り込んでいることを示し、ドルの方向感を阻害する。
  • claim: GDP成長率の低水準と消費者信頼感急落が米国景気後退リスクを高める / data: {'gdp_qoq_annualized': 1.99, 'unrate': 4.3} / evidence: GDP前期比年率が1.99%と潜在成長率付近に留まる中、ミシガン大指数が49.8まで急落した。個人消費は米GDPの約70%を占めるため、消費者マインドの悪化が実体経済に波及した場合、FRBの利下げ加速を促しドルに下落圧力がかかる。
  • claim: SKEW高水準が示すテールリスクとドル以外の安全通貨への逃避リスク / data: {'skew': 136.96, 'put_call_ratio_total': 0.95} / evidence: SKEWインデックスが136.96と依然高水準にあり、市場参加者が低確率の大幅下落に対してヘッジを積み上げていることを示す。テールリスクが顕在化した際には、円・スイスフランなどドル以外の安全通貨が選好されドルが売られるシナリオがある。
総合判断
DXYは前週比でテクニカル的に改善し、50日MA・200日MAを双方上回るポジティブな変化が生じた。CPI3.95%高止まりとFFレート3.62%の高水準が金利差優位を維持し、米長期金利の上昇（10Y: 4.57%）もドル建て資産への資本流入を促す。一方、ミシガン大消費者信頼感が49.8と典型レンジ下限を割り込んだことは今週の最大の下方リスクであり、GDP成長率の低迷と合わせると米国経済の軟着陸に対する懸念が高まりつつある。VIX3mの20.0というコンタンゴ構造も中期的な不確実性の継続を示している。テクニカル改善と金利差優位がやや優勢であり現状は「わずかに強気」と評価するが、消費者マインドの急落を考慮すると確信度は限定的である。
主なリスク
  • ミシガン大消費者信頼感の悪化が実体消費に波及し、GDP成長率が大幅に下振れした場合のドル急落リスク
  • FRBが想定以上のペースで利下げに転じた場合、金利差縮小によるドルの急落
  • VIX3mが示す中期リスクが顕在化し、円・スイスフランなどドル以外の安全通貨へのシフトが加速するシナリオ
  • 米国の財政赤字拡大や格付け引き下げ懸念がドルの基軸通貨としての信認を損なうリスク
主な触媒（カタリスト）
  • CPI再加速（4.5%超）によるFRBタカ派姿勢の強化と利下げ先送り観測の台頭
  • DXYが100の心理的節目を突破しテクニカルブレイクアウトによる買いモメンタムの加速
  • 地政学的リスクの高まりによる安全通貨としてのドル需要急増
  • 米国の消費・雇用統計が予想を上回り景気後退懸念が払拭されるシナリオ
